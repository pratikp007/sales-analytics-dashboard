# Sales Analytics Dashboard

## Project Overview
Analyzed 9,800 rows of superstore sales data using 
Python to find key business insights.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- SQL

## Key Findings
1. Technology category is #1 — $8,27,455 total sales
2. West region leads — $7,10,219 total sales  
3. California is top state — $4,46,306 total sales
4. Sean Miller is top customer — $22,638 single order
5. Canon imageCLASS 2200 is most popular product

## Charts Created
- sales_by_category.png — Sales comparison by category
- sales_by_region.png — Regional sales breakdown
- top_customers.png — Top 10 customers ranking
- monthly_trend.png — Monthly sales trend line

## How to Run
pip install pandas matplotlib seaborn
jupyter notebook sales_analysis.ipynb

Superstore sales data — 9,800 rows, 18 columns
